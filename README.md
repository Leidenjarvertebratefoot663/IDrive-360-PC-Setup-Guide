## 🧩 Troubleshooting Common Errors

Software installation often fails due to existing system conflicts. Review these solutions if you see error codes or the installer stops responding.

### Permission Denied Messages
Windows 11 features strict security controls. If you see an error regarding access, close all open programs. Disable your third-party antivirus software temporarily. Reopen your download folder and run the installer again as an administrator.

### The Installer Stops at 99 Percent
This error usually indicates a conflict with a previous version of the software. Open the Control Panel on your PC. Select Uninstall a program. Look for any remnants of IDrive in the list. Remove any older files, restart your computer, and run the setup file again.

### Network Timeouts
Cloud software requires a steady connection to verify the installer file. A unstable Wi-Fi signal causes the download or the verification to fail. Connect your PC to the router with an Ethernet cable if you experience repeated network errors during the setup sequence.

### Windows 11 Compatibility Warnings
If you receive a notification stating the app is not compatible with Windows 11, right-click the installation file. Select Properties. Select the Compatibility tab. Click the box for Run this program in compatibility mode for and choose Windows 10 from the list. Click Apply and then try to run the file again.
